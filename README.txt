Bu repodaki programlar projenin sadece belirli bir kısmıdır. Projede yönetim programı sunucuya bağlı olan seraların(client) sıcaklıklarını ve çalışma durumunu kontrol eder. Sunucuya ilgili işlem kodlarını göndererek ilgili seranın gerekli aksiyonu almasını sağlar. Sunucu programı yönetim programı ve seralar arasındaki bağlantıyı sağlar. Client programı ise serayı temsil eder, client programında seri haberleşme portlarından okunan veri okunarak ilgili seranın durumu sunucu üzerinden yönetim programına iletilir. Ekran görüntülerini inceleyerek proje hakkında daha fazla fikir sahibi olabilirsiniz.


The programs in this repo are only a specific part of the project. In the project, the management program controls the temperatures and working status of the greenhouses (client) connected to the server. It sends the relevant operation codes to the server and ensures that the relevant greenhouse takes the necessary action. The server program provides the link between the management program and the greenhouses. The client program represents the greenhouse, the data read from the serial communication ports in the client program is read and the status of the relevant greenhouse is transmitted to the management program over the server. You can have more ideas about the project by reviewing the screenshots.
